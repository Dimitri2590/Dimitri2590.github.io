## 📂 Projets Mis en Avant
Voici quelques-uns des projets détaillés sur le site :

### 1. Le jeu des 7 erreurs 🧐
Un jeu interactif réalisé pour la présentation "La Voie des Talents". Le défi : trouver les erreurs cachées dans l'interface du site.

### 2. Bikedle 🏍️
Un mini-jeu inspiré de Wordle où le joueur doit deviner un modèle de moto à partir d'une image qui se dévoile progressivement.
- *Contexte* : Projet d'évaluation scolaire.

### 3. Watchlist React 🎬
Application personnelle de gestion de films à voir, développée pour monter en compétence sur la librairie React.

## 🚀 Installation & Lancement local

Pour explorer le code source ou tester le site localement :

1.  **Cloner le dépôt :**
    ```bash
    git clone https://github.com/Dimitri2590/Dimitri2590.github.io.git
    ```

2.  **Accéder au dossier :**
    ```bash
    cd Dimitri2590.github.io
    ```

3.  **Lancer le site :**
    Ouvrez simplement le fichier `index.html` dans votre navigateur préféré.
    > _Pour une meilleure expérience (notamment pour les modules JS), l'utilisation d'une extension comme "Live Server" sur VS Code est recommandée._
